2.4 Tidy Up the Data

The tables from the report are not in a tidy format with observations in rows and variables in columns, so we tidy the data and compine tables 15 and 16 since they both relate to Nigerian states.

2.4.1 Table 4

Table 4 is already tidy. All we do here is simplify the variable names.

Calculations 
t4 <- read_excel("sources/2021/malaria-consortium-2021-table-04.xlsx",
                   col_types = c("text", "numeric", "numeric")) |>
  rename(state = 1, clusters = 2, targets = 3)

2.4.2 Table 14

For Table 14, we perform the following operations:

  1. Simplify the variable names.
  2. Create a new variable named country, and copy the data from the survey variable when n is empty.
  3. For every empty value for country, fill it with the closest non-empty value above it.
  4. Drop the rows where n is empty.
  5. Separate country names from the parenthetical notes.
  6. Delete the “)” at the end of the parenthetical notes.
  7. Separate the c_interval variable into left_ci and right_ci, then store them as numbers.
Calculations 
t_14 <- read_excel("sources/2021/malaria-consortium-2021-table-14.xlsx",
                   col_types = c("text", "numeric", "numeric", "text")) |>
  rename(survey = 1, n = 2, p_hat = 3, c_interval = 4) |>
  mutate(country = case_when(is.na(n) ~ survey,
                              TRUE ~ NA),
         country = na.locf(country)) |>
  filter(!is.na(n)) |>
  separate_wider_delim(cols = country,
                       delim = "(",
                       names = c("country", "country_note"),
                       too_few = "align_start") |>
  mutate(country_note = str_sub(country_note, end = -2)) |>
  separate_wider_delim(cols = c_interval,
                       delim = '-',
                       names = c("left_ci", "right_ci")) |>
  mutate(left_ci = as.numeric(left_ci)/100,
         right_ci = as.numeric(right_ci)/100)

2.4.3 Table 15

For Table 15, we perform the following operations:

  1. Simplify the variable names.
  2. Create a new variable named country that is equal to “Nigeria,” because all of this data is from Nigera.
  3. Create a new variable named state, and copy the data from the survey variable when n is empty.
  4. For every empty value for state, fill it with the closest non-empty value above it.
  5. Drop the rows where n is empty.
  6. Separate the c_interal variable into left_ci and right_ci, then make store them as numbers.
  7. This table relates to the Nigerian states that had 5 cycles, so create a variable to remember this.
Calculations 
t_15 <- read_excel("sources/2021/malaria-consortium-2021-table-15.xlsx",
                   col_types = c("text", "numeric", "numeric", "text")) |>
  rename(survey = 1, n = 2, p_hat = 3, c_interval = 4) |>
  mutate(country = "Nigeria",
         state = case_when(is.na(n) ~ survey,
                              TRUE ~ NA),
         state = na.locf(state)) |>
  filter(!is.na(n)) |>
  separate_wider_delim(cols = c_interval,
                       delim = '-',
                       names = c("left_ci", "right_ci")) |>
  mutate(left_ci = as.numeric(left_ci)/100,
         right_ci = as.numeric(right_ci)/100) |>
  mutate(n_cycles = "5-cycle states")

2.5 Percentage of targeted children interviewed for post-cycle surveys, Nigeria

This subsection mirrors the calculations at the bottom of GiveWell’s spreadsheet.

Calculations 
gw1516 <- t_1516 |>
  left_join(t4, join_by(state)) |>
  group_by(survey) |>
  summarise(sampled = sum(n), targeted = sum(targets)) |>
  mutate(percent_targeted_interviewed = label_percent(accuracy = 1)(sampled/targeted)) |>
  filter(survey != "EoR: cycle 5")

gw1516|>
  kable(digits = 1, format.args = list(big.mark = ","))

Note that only the state with five cycles–Kogi, Nasarawa, and Plateau–had EoC: cycle 1.

There is one discrepany between this table and GiveWell’s spreadsheet cell C82. This is that EoC: cycle 2 in the targetted column as 26900. However, this analysis does not count those targetted in Borno since there is no cycle 2 listed for Borno in Table 16 from Malaria Consortium, 2021 SMC coverage report, Burkina Faso, Chad, Mozambique, Nigeria, Togo, and Uganda.
